100 Dart Coding Challenges
Solve each of the 100 practical coding problems below using Dart. Problems are grouped by topic and increase in difficulty within each section. Try to solve each one on your own before looking up solutions!


1. Numbers & Math
	•	Find the largest number in a list.
	•	Find the smallest number in a list.
	•	Find the second largest number in a list.
	•	Find the sum of all numbers in a list.
	•	Find the average of all numbers in a list.
	•	Check if a number is prime.
	•	Find all prime numbers between 1 and 100 (Sieve of Eratosthenes).
	•	Check if a number is a palindrome (e.g. 121).
	•	Find the factorial of a number using recursion.
	•	Find the GCD (Greatest Common Divisor) of two numbers.

2. Lists & Arrays
	•	Merge two lists into one.
	•	Remove duplicate elements from a list.
	•	Reverse a list without using the built-in reverse method.
	•	Sort a list of integers in ascending and descending order.
	•	Find the intersection of two lists (common elements).
	•	Find the union of two lists (all unique elements).
	•	Rotate a list to the left by k positions.
	•	Find the missing number in a list from 1 to n.
	•	Count how many times a specific element appears in a list.
	•	Flatten a nested list into a single list.

3. Strings
	•	Count the frequency of each character in a string.
	•	Check if a string is a palindrome (e.g. 'racecar').
	•	Count the number of vowels and consonants in a string.
	•	Remove all spaces from a string.
	•	Find the most frequently occurring character in a string.
	•	Check if two strings are anagrams of each other.
	•	Reverse the words in a sentence.
	•	Count the number of words in a string.
	•	Find all duplicate characters in a string.
	•	Capitalize the first letter of every word in a string.

4. Searching & Sorting
	•	Implement a linear search to find an element in a list.
	•	Implement a binary search on a sorted list.
	•	Implement bubble sort on a list of integers.
	•	Implement selection sort on a list of integers.
	•	Implement insertion sort on a list of integers.
	•	Sort a list of strings alphabetically.
	•	Sort a list of maps by a specific key (e.g. sort users by age).
	•	Find the index of the first occurrence of an element in a list.
	•	Find the index of the last occurrence of an element in a list.
	•	Check if a list is sorted in ascending order.

5. Maps & Frequency Problems
	•	Count the frequency of each word in a sentence.
	•	Find the most frequent element in a list.
	•	Group a list of strings by their first letter.
	•	Find all elements in a list that appear more than once.
	•	Invert a map (swap keys and values).
	•	Merge two maps and sum the values for common keys.
	•	Find the top 3 most frequent words in a paragraph.
	•	Count how many times each digit (0–9) appears in a number.
	•	Given a list of names, count how many names start with each letter.
	•	Find elements that appear in one list but not the other.

6. Patterns & Logic
	•	Print a right-angled triangle star pattern of n rows.
	•	Print a pyramid star pattern of n rows.
	•	Print an inverted triangle star pattern.
	•	Print the Floyd's Triangle (numbers in a triangle pattern).
	•	Print a diamond pattern using stars.
	•	Generate the first n rows of Pascal's Triangle.
	•	Print a checkerboard pattern of size n x n.
	•	Print numbers 1 to 100, but print 'Fizz' for multiples of 3, 'Buzz' for multiples of 5, and 'FizzBuzz' for both.
	•	Print a hollow rectangle border of stars.
	•	Generate a multiplication table for a given number n.

7. Recursion
	•	Calculate the nth Fibonacci number using recursion.
	•	Calculate the power of a number (x^n) using recursion.
	•	Sum all digits of a number using recursion (e.g. 123 → 6).
	•	Find the reverse of a string using recursion.
	•	Check if a string is a palindrome using recursion.
	•	Count the number of zeros in a number using recursion.
	•	Flatten a nested list using recursion.
	•	Find the sum of all elements in a nested list using recursion.
	•	Print all permutations of a string using recursion.
	•	Solve the Tower of Hanoi problem for n disks.

8. OOP Challenges
	•	Create a Student class with name, age, and grade. Add a method to check if the student passed (grade >= 50).
	•	Create a BankAccount class with deposit, withdraw, and balance methods.
	•	Create a Stack class with push, pop, peek, and isEmpty methods using a List.
	•	Create a Queue class with enqueue, dequeue, and peek methods using a List.
	•	Create a Shape base class and extend it with Circle and Rectangle. Override an area() method.
	•	Create a Product class and write a method to find the most expensive product in a list.
	•	Create a Library class that manages a list of Books and can add, remove, and search by title.
	•	Create a Calculator class with add, subtract, multiply, and divide methods. Handle division by zero.
	•	Create a Person class with a compareTo method and sort a list of Person objects by age.
	•	Create a simple linked list with Node class and implement insert and print methods.

9. String & Number Conversion
	•	Convert a string of digits to an integer without using int.parse().
	•	Convert an integer to its binary representation as a string.
	•	Convert a binary string back to a decimal integer.
	•	Convert a number to its Roman numeral representation.
	•	Convert Roman numerals back to a decimal integer.
	•	Check if a number is an Armstrong number (e.g. 153 = 1³+5³+3³).
	•	Find the sum of digits of a number until it becomes a single digit.
	•	Check if a number is a perfect number (sum of divisors equals the number).
	•	Convert a temperature from Celsius to Fahrenheit and vice versa.
	•	Given a number, return the individual digits as a list.

10. Real-world Mini Problems
	•	Given a list of scores, print the grade letter for each score (A/B/C/D/F).
	•	Find the longest word in a sentence.
	•	Find the shortest word in a sentence.
	•	Remove all vowels from a string.
	•	Given a list of integers, separate them into even and odd lists.
	•	Find all pairs in a list that sum to a given target.
	•	Given a list of names, remove names shorter than 4 characters.
	•	Find the longest common prefix among a list of strings.
	•	Given a sentence, count how many times a specific word appears.
	•	Given a list of numbers, return only the unique elements while preserving order.
